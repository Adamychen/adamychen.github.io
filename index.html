Ejercicio 1

Diseñar programas GOTO que, sin usar macros (salvo quizás GOTO L), calculen las funciones:

\(f_1(x) = 3x \quad \)
\(f_2(x) = \begin{cases} 1 & \text{si } x \text{ es par} \\ 0 & \text{si } x \text{ es impar} \end{cases} \)
\(f_3(x) = \begin{cases} 1 & \text{si } x \text{ es par} \\ \uparrow & \text{si } x \text{ es impar} \end{cases} \)
Solución \(f_1\)>

\(f_1\)

El siguiente programa \(P_1\) calcula la función \(f_1\)

\(\quad Z_1 \leftarrow Z_1 + 1\)

\(\quad Z_1 \leftarrow Z_1 + 1\)

\(\quad Z_1 \leftarrow Z_1 + 1\)

\(\quad\) IF X \( \neq 0\) GOTO A

\(\quad Z_5 \leftarrow Z_5 + 1 \)

\(\quad\) IF \(Z_5 \neq 0\) GOTO E

[A] \( Y \leftarrow Y + 1\)

\(\quad\) \(Z_2 \leftarrow Z_2 + 1\)

\(\quad\) \(X \leftarrow X - 1\)

\(\quad\) IF \(X \neq 0\) GOTO E

\(\quad Z_1 \leftarrow Z_1 - 1\)

\(\quad\) IF \(Z_1 \neq 0\) GOTO B

\(\quad Z_1 \leftarrow Z_1 - 1\)

\(\quad\) IF \(Z_1 \neq 0\) GOTO B

\(\quad Z_5 \leftarrow Z_5 + 1 \)

\(\quad\) IF \(Z_5 \neq 0\) GOTO E

[B] \( Z_2 \leftarrow Z_2 - 1\)

\(\quad\) \(X \leftarrow X + 1\)

\(\quad\) IF \(Z_2 \neq 0\) GOTO B

\(\quad\) IF \(X \neq 0\) GOTO A

[E] \(X \leftarrow X \)

Ejercicio 2

\(g_2(x_1, x_2) = \begin{cases} 1 & \text{si } x_1 \leq x_2 \\ 0 & \text{si } x_1 > x_2 \end{cases}\)

\(g_2(x_1, x_2) = \begin{cases} 1 & \text{si } x_1 \leq x_2 \\ 0 & \text{si } x_1 > x_2 \end{cases}\)

Ejercicio 13

Sea \(GOTO_A\) el lenguaje definido como \(GOTO\) pero cuyas intrucciones (etiquetadas o no) son:

\[ V \leftarrow V' \] \[ V \leftarrow V + 1 \] \[ \text{IF} \quad V \neq V' \quad \text{GOTO L} \]

que tienen interpretación obvia. Probar que una función \(f\) es \(GOTO_A\)-computable si y sólo si \(f\) es \(GOTO\)-computable

Solución

Si \(f\) es \(GOTO_A\)-computable entonces \(f\) es \(GOTO\)-computable ya que:

Sea \(P\) el programa obtenido a partir de \(Q\) interpretando las instrucciones de \(Q\) como macros de \(GOTO\). Entonces \([[P]] = f\), por lo que \(f\) es \(GOTO_A\)-computable

Tenemos entonces que si \(f\) es \(GOTO_A\)-computable \(\rightarrow f\) es \(GOTO\)-computable

Basta probar que si \(\rightarrow f\) es \(GOTO\)-computable entonces \(f\) es \(GOTO_A\)-computable

Tenemos que probar que las instrucciones de \(GOTO\) que no están en \(GOTO_A\) son macros de \(GOTO_A\)

\[ \text{IF } V \neq 0 \text{ GOTO L} \] \[ V \leftarrow V \] \[ V \leftarrow V - 1 \]

Para la macro \(\text{IF } V \neq 0 \text{ GOTO L}\)

\(\quad Z_1 \leftarrow X_1 = X_2 \)

\(\quad\)IF \(Z_1 \neq 0\) GOTO L

\(\quad Z_1 \leftarrow Z_1 + 1\)

\(\quad\)GOTO E

[L] \(Z_1 \leftarrow Z_1 - 1\)
